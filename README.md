##### Rukaml --- miniML compiler

In future it may become closer to real OCaml compiler but now we have only this:

* HM Type checker
* functions
* tuple
* primitives (like `print_int`)
* Simple copying GC

Backends:

* RISC V (most mature one)
* AMD64
* LLVM
